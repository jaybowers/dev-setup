# Workstation setup

A simple bash script to set up a new workstation.

## Limitations

1. Very opinionated
2. Only tested with Linux Mint 18.1

## Quick Start

```
    apt-get install git \
        && git clone git@github.com:jaybowers/dev-setup.git \
        && cd desktop-setup \
        && ./setup.sh <other desktop>
```
